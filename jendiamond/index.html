I was working on an animation for a planetarium dome when I learned some Ruby so I could complete my idea. That's when I realized that programming was the missing link to my future. I started taking more programming classes and attended Railsbridge. I ended up so inspired I started tutoring at Railsgirls, Railsbridge, RubyKids and started a weekly study group.

This summer I was part of Rails Girls Summer of Code Team Bundler. I learned so much and I even helped make some improvements to Bundler. I am going to continue learning and sharing what i learn.

I will be posting my tools, goals, progress, and achievements to this blog so others can learn from what I have learned and so I can refresh my own memory.
